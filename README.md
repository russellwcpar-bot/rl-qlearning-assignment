# rl-qlearning-assignment
Module 9 assignment 
The training process begins with the agent exploring its environment through a trial and error approach. The environment resets at the start of each episode allowing the agent to begin in a fresh state, and test various actions. Using the epsilon greedy strategy the agent initially prioritizes exploration. As training progresses and the epsilon value decays, it shifts toward exploitation, choosing the action with the highest known Q value to maximize its efficiency.  The agent executes it and observes the resulting reward and new state. This feedback is used to update the table using the Bellman Equation, which calculates the difference between the current predicted reward and the actual reward received plus the discounted potential of the next state. This  step  allows the agent to teach the taxi how to navigate while avoiding penalties.  After the training episodes are complete, the agent's performance is tested using a greedy policy to ensure it has learned the optimal path. By plotting a moving average of total rewards. the graph shows a sharp rise as the agent stops making penalized moves and begins successfully completing runs,and leveling off once it has figured out the map.
